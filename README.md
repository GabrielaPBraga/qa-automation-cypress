# 🧪 QA Automation - Cypress

Projeto de automação de testes E2E utilizando Cypress.

## 🚀 Tecnologias
- Cypress
- JavaScript

## 📁 Estrutura do projeto

```
cypress/
  e2e/
    login.cy.js
    produtos.cy.js
    checkout.cy.js
  pages/
    loginPage.js
    produtosPage.js
    carrinhoPage.js
    checkoutPage.js
  fixtures/
    user.json
  support/
    commands.js
```
